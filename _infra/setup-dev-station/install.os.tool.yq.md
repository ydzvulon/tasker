# Install Act. shell.query.tool.yq4

> `linux`

```bash
wget https://github.com/mikefarah/yq/releases/download/v4.16.1/yq_linux_amd64
chmod +x yq_linux_amd64
sudo mv yq_linux_amd64 /usr/local/bin/yq
```
